Contributing
============

We welcome contributions to tf-kernel! This page provides guidelines for contributing.

How to Contribute
-----------------

1. **Fork the repository** on GitHub
2. **Create a feature branch** (``git checkout -b feature/amazing-feature``)
3. **Make your changes**
4. **Run tests** (``make test``)
5. **Format your code** (``make format``)
6. **Commit your changes** (``git commit -m 'Add amazing feature'``)
7. **Push to your fork** (``git push origin feature/amazing-feature``)
8. **Open a Pull Request**

Reporting Bugs
--------------

Please use the `GitHub Issues <https://github.com/YOUR_ORG/tf-kernel/issues>`_ to report bugs.

When filing a bug report, please include:

- System information (OS, CUDA version, GPU architecture)
- Python and PyTorch versions
- Steps to reproduce
- Expected vs actual behavior
- Error messages or stack traces

Requesting Features
-------------------

Feature requests are also welcome via GitHub Issues. Please:

- Use a clear and descriptive title
- Provide a detailed description
- Explain why the feature would be useful
- Include examples of usage if possible

Code of Conduct
---------------

Please note that this project is released with a `Contributor Code of Conduct <https://github.com/YOUR_ORG/tf-kernel/blob/main/CODE_OF_CONDUCT.md>`_.
By participating in this project you agree to abide by its terms.
